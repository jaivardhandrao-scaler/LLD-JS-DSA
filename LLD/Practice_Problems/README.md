# LLD Practice Problems

## How to use

1. Read the `PROBLEM.md` in each folder
2. Create your Java classes as `.java` files in the same folder
3. Ask me to score -- I'll read your files and give you a score out of 10 with feedback

## Scoring Rubric (same for all)

| Criteria | Points |
|----------|--------|
| **Correctness** -- code compiles, logic is right | 3 |
| **OOP/Pattern usage** -- correct application of the required pattern | 3 |
| **SOLID compliance** -- no violations | 2 |
| **Encapsulation** -- proper access modifiers, validation | 1 |
| **Clean code** -- readable, no dead code, sensible names | 1 |
| **Total** | **10** |
